## Run Locally

Clone the project

```bash
  git clone https://github.com/sidkhillon/CS35LBankProject
```

Go to the project directory

```bash
  cd CS35LBankProject
  cd bank
```

Install dependencies

```bash
  npm install
```

Start the server locally

```bash
  npm start
```
